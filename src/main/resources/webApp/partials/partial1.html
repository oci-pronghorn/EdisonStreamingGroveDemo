All the following steps are done on the embedded device with Zulu.
  1. Data generation/capture
  2. Publish to the MQTT Broker
  3. Subscribe to events with Paho
  4. Pronghorn sends events over WebSockets with Netty
  5. Netty also hosts the static pages for the AngularJS SPA

Your browser runs the Single Page Application(SPA) and displays the data using smoothie charts.
